Изначальный список людей, допущенных к избиранию:

"Ковалёв Андрей Сергеевич, 1234, 123456"
"Виноградова Екатерина Евгеньевна, 2345, 234567"
"Королёв Алексей Игоревич, 3456, 345678"

При вводе данных, отсутствующих в списке, программа не зачтёт голос,
оповестив об этом надписью "Вам не нужно голосовать".

Список можно изменить, нажав на "Импорт списка избирателей" в настройках.
Списком избирателей может быть любой текстовый файл, содержаший на каждой строке ФИО,
серию и номер паспорта избираемого через запятую.
