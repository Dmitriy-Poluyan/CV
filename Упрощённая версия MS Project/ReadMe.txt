Веб-приложение, позволяющее пользователю создать график работ и задач, выполняемых в проекте,
аналогично диаграмме Ганта в программе Microsoft Project

График состоит из визуально представляемых задач.
Длина задачи показывает её длительность в днях.
Длину можно изменять, потянув за правый край задачи.
Отступ задачи от начала графика показывает
время начала выполнения задачи от начала проекта в днях.
Отступ можно изменять, потянув за центр задачи.
